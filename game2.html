Sporky needs to travel to the market to purchase a new scarf. My legs cannot prolong for long distances so, I must travel in another means of transportation. The available options are a dazzling yellow car, a strong, cute and resilient camel, a shiny but fiesty scooter or I can just be tested to my limits and walk there.

Analysing the code, decide which option will be the most time efficient based on the output. 
        public class MyClass {
            public static void main(String[] args) {
                int carSpeed = 30;
                int camelSpeed = 60;
                int scooterSpeed = 20;
                int walkingSpeed = 5;
            
                int carTime;   
                int camelTime;
                int scooterTime;
                int walkingTime;
            
                int distance = 60;
            
                // TIME = DISTANCE/SPEED;
                carTime = distance/carSpeed;
                camelTime = distance/camelSpeed;
                scooterTime = distance/scooterSpeed;
                walkingTime = distance/walkingSpeed;
            
            
                System.out.println("Sporky travelled for: " + distance + "km");
                    
                System.out.print("Car: " + carTime );
                System.out.println(" hour/s");
            
                System.out.print("Camel: " + camelTime);
                System.out.println(" hour/s");
            
                System.out.print("Scooter: " + scooterTime);
                System.out.println(" hour/s");
            
                System.out.print("Walking: " + walkingTime);
                System.out.println(" hour/s");
            
            }
        }
